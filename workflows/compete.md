# Competitive Analysis Workflow

## Input
Product space, competitor names, or both. If only a space is given, ask: "Name 2–4 competitors you want to analyze."

Do NOT assume web access. Only use information the user provides or that is in existing project files.

Save to `docs/compete-[space-kebab-case].md`:
